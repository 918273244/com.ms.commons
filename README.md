com.ms.commons
==============
### Summer框架(对Spring MVC 的深度封装)

com.ms.commons提供了一个功能比较全面的summer框架及其依赖包，
同时放了webSiteDemo工程的依赖包，
也是一个web工程必备的常用包;

主要有:
* summer框架
* cookie支持
* fasttext 页面违禁词校验
* security 安全包配置信息
* solr 搜索支持
* nisa 基于mina实现的集中式配置中心
* standalone 定时任务支持
* udas 缓存服务支持
* db 数据库持久化支持
* summer 框架核心依赖core
* test 测试相关支持
* message 邮件短信支持
* weixin 微信支付相关接口封装
